# 雅思词汇闪卡 (IELTS Vocabulary Flashcards)

一个简洁优雅的雅思词汇学习工具，采用闪卡形式帮助记忆。

## 功能特点

- 📚 **分类学习** - 按词汇难度分类（基础、进阶、高级）
- 🎴 **闪卡模式** - 正面单词，背面释义，点击翻转
- 🔊 **发音支持** - 点击播放英式/美式发音
- 📊 **学习进度** - 本地保存学习进度
- 🎨 **主题切换** - 多种配色主题可选
- 📱 **响应式设计** - 支持桌面和移动设备

## 使用方法

1. 直接在浏览器中打开 `index.html`
2. 选择词汇分类开始学习
3. 点击卡片翻转查看释义
4. 使用键盘方向键或按钮切换单词

## 技术栈

- 纯前端实现（HTML + CSS + JavaScript）
- 无需后端，完全本地运行
- 数据存储在浏览器 LocalStorage

## 数据来源

词汇数据整理自雅思核心词汇表。

## 注意事项

- 学习进度保存在浏览器本地存储中，清除缓存会丢失进度
- 建议定期导出备份学习进度
- 部分词汇词性可能存在识别误差
